## Feature Highlights

### ✅ QR Code Scanning
- Real-time camera scanning with instant detection
- Visual scanning guide overlay
- Haptic feedback on successful scan

### ✅ Image Upload Option
- Upload QR code images from gallery
- Alternative scanning method when camera unavailable

### ✅ Smart Camera Handling
- Automatic permission requests
- Auto-fallback to upload mode if camera denied/unavailable
- Web platform automatically uses upload mode

### ✅ URL Security & Validation
- Only accepts secure URLs (http://, https://, www.)
- Automatic URL formatting
- Invalid URL detection with clear error messages

### ✅ In-App Browser
- Opens scanned URLs within the app
- Full web browsing experience
- Loading indicators and refresh functionality
- No need to leave the app

### ✅ Cross-Platform
- Android (API 24+)
- iOS
- Web browser support

### ✅ Modern UI/UX
- Clean Material Design 3 interface
- Intuitive navigation
- Seamless user experience
